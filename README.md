# Star Forge

A browser-based sci-fi idle game.

## Play the latest build

Once GitHub Pages is enabled for this repository, the latest version will always be available here:

**https://omniflesh.github.io/StarForge/**

If GitHub Pages has not been enabled yet:

1. Go to **Settings → Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select **main** and **/(root)**.
4. Save.

After a minute or two, the link above will always load the latest build from the `main` branch.
